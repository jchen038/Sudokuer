== Sudokuer. A Sudoku Solver

- Versions
Ruby 2.0.0 +
Rail 4.2.0


- Setup Instructions

1. git clone repo
2. bundle install
3. rake db:create
4. rake db:migrate
5. rails s
6. open browser and go localhost:3000
